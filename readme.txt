How to set up this Test Script

1.- Make sure you have appium server/client installed 
2.- Initiate an appium server session
3.- Connect Dr. Strange with G7 app in automatic mode
4.- Open script and set the # of desired trials for your tests
4.- Leave open G7 app and run the Script
